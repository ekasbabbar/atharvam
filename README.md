# 🌱 Atharvam — Green Activated Carbon from Biomass

**Turning India’s Residue into India’s Energy**

---

## 🚀 Overview

Atharvam is a sustainability-driven materials startup focused on producing **high-performance activated carbon** from agricultural waste using a **green synthesis process**.

The project was developed as part of **Startup Sprint 2026**, targeting India’s growing demand for energy storage and filtration materials.

## ❗ Problem

* India imports **~$98M+ activated carbon annually**
* ~500M tonnes of agricultural waste generated yearly
* Biomass waste → methane emissions (≈80× CO₂ impact)
* Conventional carbon production:

  * Uses **fossil fuels**
  * Requires **corrosive chemicals (KOH, acids)**
  * Produces toxic wastewater

---

## 💡 Solution

We convert **tea waste + banana biomass** into activated carbon using a **bio-chemical activation process**:

* 🌿 Tea waste → carbon precursor
* 🍌 Banana extract → natural activating agent (K₂CO₃)
* 🔥 Thermal carbonization → porous carbon

👉 Eliminates toxic chemicals while maintaining industrial-grade performance.

## 🔬 Key Innovation

* Replaces **KOH activation** with **banana-derived potassium compounds**
* Enables:

  * ✅ Lower environmental impact
  * ✅ Reduced cost
  * ✅ Safer production
